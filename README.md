# My Personal Project
To record the retrieving and adding of money with their specific time and purpose. This app is for people who want to balance their spending and control budget better.

For me, when I spend money, I don’t really feel it, so my budget control is not reasonable. I often realize it only after I already spent too much. A lot of my friends are also like this. So I think this application will be super helpful and useful in real life.

As a user, when I start the application, I want to be given the option to load my ExpenseRecorder from file.
